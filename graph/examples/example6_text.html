Example 6

This plot demonstrates the use of the Contour Class. The contour levels are either manually set or automatically set.

The Contour class is an extension of the G2Dint class and therefore has all the interactive control of that class. It also has the following extra event handlers

Keypress l
The plot is redrawn Without the contour labels.
Keypress L
The plot is redrawn With the contour labels.
